Options

Ignore Elements Matching...

Enter a list of selectors (one per line). The generator will ignore all elements that match the following selectors.

Preferred Assertion Property

The generator will choose to assert against this property if it exist. If it does not, another property will be chosen instead.

Ignore Selector Types

Selectors of the chosen types will be ignored, and will not show up in the "assert" selector field.

Selector Attributes

Attributes of the chosen types will be appended to selectors.

Preferred Hierarchy Cardnality

The generator will attempt to find a selector with the hierarchy cardnality greater than or equal to the chosen value. The hierarchy cardnality is the number of elements in the selector.
If a cardnality of the selected value doesn't exist, the most appropriate will be chosen instead.

The suite and dropdown fields below will determine the suite and module the list is saved under.
Warning: The "name" field is tied to the action (test, screen, etc.) filename. Changing it will save loaded actions as a new file.
